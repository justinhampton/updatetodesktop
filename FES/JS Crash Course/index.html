Data Types & Variables

Data types help us organize data

  • String (text) - inside quotes, any character, text numbers emojis etc, can be added together with separate quotes and + ex: 'hello' + 'world'. String indexing starts from 0, 1, 2, etc. 'Justin'[0] will locate the first character of the string
  • Number (just a number)
  • Boolean (true or false)
  • Undefined (nothing, value was never set)
  • Null (value of nothing, specifically set)

Variables

  • var (outdated, old code), let (usable everywhere, can use to define your variables let fullName = 'Justin Hampton', use camelCase), and const (almost the same as let but you can't reassign like you can with let, const is a global constant that you can't change. think of it like ID, you only want to use it once for something that wont be changing.

Equality

== and ===

  • == double equals checks the values
  • === triple equals checks the type and the values
  • the opposite is to add a !, so !== and !=== because the ! means 'not'

Conditionals

You can have an if without an else, but you can't have an else without an if

  • == equal values
  • === equal values and types
  • != not equal
  • !== not equal value and equal type
  • > greater than
  • < less than
  • >= greater than or equal to
  • <= less than or equal to

Logical Operators

  • && and checks if the left and the right side are true.
    • true && true -> true
    • true && false -> false
    • false && true -> false
    • false && false -> false
  • || checks if the left or right side is true
    • true || true -> true
    • true || false -> true
    • false || true -> true
    • false || false -> false

Falsy values, considered false when converted to a boolean

  • undefined
  • null
  • NaN
  • 0
  • ""
  • false

Truthy values, considered true when converted to boolean

  • letters, numbers, all strings
  • empty arrays
  • objects

for an if statement to run it needs to be truthy. !!"x" in the console will let you know if it's truthy or falsy

Ternary Operators, shortcut for an if else condition

isObese ? 'unhealthy' : 'healthy'

subscried ? 'show video' : 'hide video'
