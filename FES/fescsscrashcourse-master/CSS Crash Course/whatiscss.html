What is CSS?

CSS stands for Cascading Stylesheets
Like HTML, it is not a programming language, it is a Stylesheet/Styling Language
It's used for website layout and design
It can be extended with Sass/Less
ID's should be unique, classes can be reused. Some people only use classes so you don't reuse ID's.

3 Methods for Adding CSS

  • Inline CSS: Directly in the html element (NO!)
  • Internal CSS: Using style tags within a single document
  • External CSS: Linking an external .css file

Colors in CSS

  • Color names
  • HTML5 Color Names
  • Hexadecimal # 123213
  • RGB

Positioning in CSS

  • Static: renders documents in order
  • Relative: relative to position, top bottom etc
  • Absolute: target anything in position in relative
  • Fixed: fixed
  • Initial: always where the initial is
  • Inherent: inherent
